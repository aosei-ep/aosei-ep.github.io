什么是动态规划

动态规划算法通常用于求解具有某种最优性质的问题。在这类问题中，可能会有许多可行解。每一个解都对应于一个值，我们希望找到具有最优值的解。

其基本思想也是将待求解问题分解成若干个子问题，先求解子问题，然后从这些子问题的解得到原问题的解。

746. 使用最小花费爬楼梯

假设数组cost的长度为 n ，则 n 个阶梯分别对应下标 0 到 n-1，楼层顶部对应下标 n，问题等价于计算达到下标 n 的最小花费

由于可以选择下标 0 或 1 作为初始阶梯，因此有 dp[0] = dp[1] = 0

当2 ≤ i ≤ n 时，可以从下标为 i - 1 或者 i - 2 获取花费的金额，选取两者中最小的那个加上当前已花费的金额

Java

class Solution {
    public int minCostClimbingStairs(int[] cost) {
        int prev = 0, curr = 0;
        for (int i = 2; i <= cost.length; i++) {
            int next = Math.min(curr + cost[i - 1], prev + cost[i - 2]);
            prev = curr;
            curr = next;
        }
        return curr;
    }
}

Python

class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        prev = 0
        curr = 0
        for i in range(2, len(cost) + 1):
            minTag = min(curr + cost[i - 1], prev + cost[i - 2])
            prev = curr
            curr = minTag
        return curr

213. 打家劫舍 II

首先考虑最简单的情况。如果只有一间房屋，则偷窃该房屋，可以偷窃到最高总金额。如果只有两间房屋，则由于两间房屋相邻，不能同时偷窃，只能偷窃其中的一间房屋，因此选择其中金额较高的房屋进行偷窃，可以偷窃到最高总金额。

当房间数超过两间，这我们可以按照不偷第一间和不偷最后一间两种情况进行区分。

  • 不偷第一间，则下标范围为 (0, n - 2)
  • 不偷最后一间，则下标范围为(1, n - 1)

在范围内查找最大值即可

Java

class Solution {
    public int rob(int[] nums) {
        int length = nums.length;
        if (length == 1) {
            return nums[0];
        } else if (length == 2) {
            return Math.max(nums[0], nums[1]);
        }
        return Math.max(robFlag(0, length - 2, nums), robFlag(1, length - 1, nums));
    }

    public int robFlag(int start, int end, int[] nums) {
        int curr = nums[start], max = Math.max(nums[start], nums[start + 1]);
        for (int i = start + 2; i <= end; i++) {
            int temp = max;
            max = Math.max(curr + nums[i], max);
            curr = temp;
        }
        return max;
    }
}

740. 删除并获得点数

每次操作中，选择任意一个 nums[i] ，删除它和所有等于 nums[i] - 1 和 nums[i] + 1 的元素，最后获得 nums[i] 的点数。

这道题将所有数字的点数相加后，会变成打家劫舍问题。因为nums[i]的范围为1 ~ 10000，则我们可以创建一个长度为10000的数组，然后因为相邻数字会被删除，可看做相邻数字不能被打劫

Java

class Solution {
    public int deleteAndEarn(int[] nums) {
        if (nums.length == 1) return nums[0];
        int[] trans = new int[10001];
        for (int num : nums) {
            trans[num] += num;
        }
        int p = 0, q = trans[1], max = 0;
        for (int i = 2; i < trans.length; i++) {
            max = Math.max(q, p + trans[i]);
            p = q;
            q = max;
        }
        return max;
    }
}

918. 环形子数组的最大和

Kadane 算法

这题一共有两种情况（也就是相当于比53题多了一种最大子数组和是首尾连接的情况）

  • 第一种情况：这个子数组不是环状的，就是说首尾不相连。
  • 第二种情况：这个子数组一部分在首部，一部分在尾部，我们可以将这第二种情况转换成第一种情况

如下图：

所以这最大的环形子数组和 = max(最大子数组和，数组总和-最小子数组和)

class Solution {
    public int maxSubarraySumCircular(int[] nums) {
        int sum = 0, maxSum = nums[0], minSum = nums[0], currMax = 0, currMin = 0;
        for (int num : nums) {
            currMax = Math.max(currMax + num, num);
            currMin = Math.min(currMin + num, num);
            maxSum = Math.max(currMax, maxSum);
            minSum = Math.min(currMin, minSum);
            sum += num;
        }
        return maxSum > 0 ? Math.max(maxSum, sum - minSum) : maxSum;
    }
}

1014. 最佳观光组合

稍微给这个公式变形成 A[i]+i+A[j]-j ，这样就可以看成是左 A[i]+i 和右 A[j]-j 两部分和的最大值。

随着遍历数组，我们对两部分和取最大值，并且若当前的值减下标对之和比之前更大，我们就更新left部分的值即可

Java

class Solution {
    public int maxScoreSightseeingPair(int[] values) {
        int tmp = values[0], ans = Integer.MIN_VALUE;
        for (int i = 1; i < values.length; i++) {
            ans = Math.max(ans, tmp + values[i] - i);
            tmp = Math.max(tmp, values[i] + i);
        }
        return ans;
    }
}

309. 最佳买卖股票时机含冷冻期

dp[0][1] 只影响 dp[1][2] ，而 dp[1][2] 又是由 dp[0][1] 和 dp[0][2] 共同决定的，而 dp[0][2] 如上文所说，肯定就是 0 。 因此 dp[1][2] = Math.max(dp[0][1], dp[0][2]); ，只要 dp[0][1] ≤ 0 都是不会造成影响的

Java

class Solution {
    public int maxProfit(int[] prices) {
        int len = prices.length;
        if (len == 0) return 0;

        // dp[i][0]: 持有股票
        // dp[i][1]: 不持有股票，本日卖出，下一天为冷冻期
        // dp[i][2]: 不持有股票，本日无卖出，下一天非冷冻期
        int[][] dp = new int[len][3];
        dp[0][0] = -prices[0];
        for (int i = 1; i < len; i++) {
            // 持有股票: 1.昨天持有股票 2.本日买入（条件：昨天不持有）
            dp[i][0] = Math.max(dp[i - 1][0], dp[i - 1][2] - prices[i]);

            // 本日卖出: (条件:昨天持有)
            dp[i][1] = dp[i - 1][0] + prices[i];

            // 不持有，不卖出: 1.昨天卖出，不持有 2.昨天没卖出，但也不持有
            dp[i][2] = Math.max(dp[i - 1][1], dp[i - 1][2]);
        }

        // 最后一天持有股票不会有收益
        return Math.max(dp[len - 1][1], dp[len - 1][2]);
    }
}

空间优化

class Solution {
    public int maxProfit(int[] prices) {
        int len = prices.length;
        if (len == 0) return 0;

        // dp0: 持有股票
        // dp1: 不持有股票，本日卖出，下一天为冷冻期
        // dp2: 不持有股票，本日无卖出，下一天非冷冻期
        int dp0 = -prices[0], dp1 = 0, dp2 = 0;
        for (int i = 1; i < len; i++) {
            // 持有股票: 1.昨天持有股票 2.本日买入（条件：昨天不持有）
            int d0 = Math.max(dp0, dp2 - prices[i]);

            // 本日卖出: (条件:昨天持有)
            int d1 = dp0 + prices[i];

            // 不持有，不卖出: 1.昨天卖出，不持有 2.昨天没卖出，但也不持有
            int d2 = Math.max(dp1, dp2);

            dp0 = d0;
            dp1 = d1;
            dp2 = d2;
        }

        // 最后一天持有股票不会有收益
        return Math.max(dp1, dp2);
    }
}

714. 买卖股票的最佳时机含手续费

  • sell: 找到最佳卖出价格
  • buy: 找到最佳买入价格

Java

class Solution {
    public int maxProfit(int[] prices, int fee) {
        int buy = -prices[0], sell = 0;
        for (int i = 1; i < prices.length; i++) {
            sell = Math.max(sell, buy + prices[i] - fee);
            buy = Math.max(buy, sell - prices[i]);
        }
        return sell;
    }
}
